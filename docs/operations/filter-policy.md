# Filter Policy

This filter policy is intentionally conservative. It excludes files that are reinstallable or regenerated by build tools, while keeping files that represent real work.

## Include

- `data/`
- `datasets/`
- `models/`
- `checkpoints/`
- `*.pt`, `*.pth`, `*.ckpt`, `*.onnx`
- `.env`
- lockfiles such as `package-lock.json`, `pnpm-lock.yaml`, `poetry.lock`, `Cargo.lock`
- notebooks
- experiment logs and results

## Exclude

```text
**/.git/**
**/node_modules/**
**/.venv/**
**/venv/**
**/dist/**
**/build/**
**/out/**
**/.next/**
**/.nuxt/**
**/.svelte-kit/**
**/.cache/**
**/.pytest_cache/**
**/.mypy_cache/**
**/.ruff_cache/**
**/__pycache__/**
**/*.pyc
**/*.pyo
**/*.o
**/*.obj
**/*.so
**/*.dylib
**/*.dll
**/*.class
```
